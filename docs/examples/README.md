# Examples

Small example instruction files used to demonstrate the current `agents-md-xray` rules.

## Files

- [`good-minimal.md`](good-minimal.md): a compact, safe instruction file with setup, test, style, safety, and PR expectations.
- [`bad-dangerous.md`](bad-dangerous.md): a short unsafe example that should trigger dangerous-instruction checks.

## Notes

These examples are intentionally small and synthetic.

They only document behavior available in the current CLI. They do not imply support for SARIF, PR comments, auto-fix, MCP scanning, dashboards, gateways, or hosted services.
